# JBL Clone Project

This is a **JBL clone website** project featuring a responsive design and developed using **HTML** and **SCSS**. The website is crafted to mimic the style and layout of the JBL official site, offering a clean and modern user experience across various devices.

## Features

- **Responsive Design**: Optimized layout for seamless viewing on mobile, tablet, and desktop devices.
- **SCSS Styling**: Modular SCSS for organized and easily maintainable code.
- **Product Display**: Sections dedicated to showcasing JBL products with images, descriptions, and pricing information.

## Technologies Used

- **HTML5**: For structuring web content.
- **SCSS**: Used to create an enhanced and modular CSS for cleaner styling.
